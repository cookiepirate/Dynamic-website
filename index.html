House Keeping
Asset

An asset can be an object like chair, table, carpet, soap dispenser, coffee machine, networking, wiring, generator, electricals or a place like 1st floor, rest room, meeting rooms.

Add Asset
Task

Task is an activity that is performed on an asset. Multiple tasks can be performed on an asset. Like vaccuming the floor, mopping the floor or clean the soap dispenser, refilling the soap dispenser. Each task has a frequency namely hourly, daily, weekly, monthly, yearly.

Add Task
Admin

Creates and manages assets in the system. Generates and updates tasks for each such asset. Admin also manages the workers available in the system.

Sign in as Admin
Workers

A Worker is someone who performs the Task. A worker can perform multiple tasks based on the skillset.
